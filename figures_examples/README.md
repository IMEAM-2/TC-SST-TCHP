Example Figures

This folder contains selected example figures of historical tropical cyclone SST and OHC/TCHP conditions.

These figures are intended for visual reference and case review only. They do not cover all storms in the tabular dataset, and they should not be treated as a uniform quantitative dataset.

The complete figure archive is provided separately in the GitHub Releases page as a zip file.

Data sources used for these figures may include Mercator Ocean products, SODA ocean reanalysis, REMSS products, NOAA OISST, and related ocean datasets. Since these products differ in spatial resolution, temporal coverage, and data processing methods, the figures are mainly suitable for qualitative comparison and case-based illustration.

示例图像

本文件夹保存部分历史热带气旋个例的 SST、OHC/TCHP 示例图像。

这些图像主要用于个例检视和可视化参考，并不覆盖表格数据中的全部系统，也不应被视为统一的定量统计数据集。

完整图像包体积较大，已作为 zip 文件单独放在 GitHub Releases 页面中。

相关图像的数据来源可能包括 Mercator Ocean、SODA 海洋再分析、REMSS、NOAA OISST 以及其他海洋资料。由于不同资料的时空分辨率、时间覆盖和处理方法不同，这些图像更适合作为定性对比和个例展示。
